Project Details

  • Tech Stack: Python (Flask), HTML, CSS, Bootstrap, JavaScript, Firestore
  • GitHub URL: Project Link

Studify is a centralized repository for students to access and download study materials, including notes and previous year question papers.

Featuring a user-friendly interface, Studify allows students to view and revisit archived lectures for better understanding.

Additionally, integrated tools for calculating and evaluating marks provide students with a quick and accurate way to assess their academic performance.
